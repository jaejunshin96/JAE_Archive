# CLAUDE.md - Rooviz (Artist Version)

📌 WHAT: 프로젝트 개요
- [cite_start]성격: 오직 작업물의 퀄리티와 감각을 기록하는 아티스트 스타일 포트폴리오 아카이브[cite: 31].
- [cite_start]핵심 가치: 비움(Zen), 기록(Archive), 본질(Essence).
- [cite_start]기술 스택: Next.js (App Router), Tailwind CSS, GSAP/Framer Motion.

🎯 WHY: 핵심 목적
- [cite_start]신뢰 증명: 설명이 아닌 시각적 완성도로 '장인 정신'을 증명.
- [cite_start]몰입감: 사용자가 웹사이트를 탐색하는 행위 자체가 하나의 전시 관람이 되도록 구현.

🛠️ HOW: 구현 및 검증 규칙
- 디자인 원칙: "Less is More". [cite_start]불필요한 버튼, 화려한 UI 요소를 배제하고 이미지와 타이포그래피 중심으로 구성.
- 애니메이션: 속도보다는 부드러움(Ease-in-out)에 집중. GSAP ScrollTrigger를 적극 활용.
- [cite_start]빌드/검증: `npm run build`를 통해 타입 오류 및 린트 체크 필수 수행.

🚫 Constraints (제한 사항)
- 과도한 마케팅 문구나 영업용 버튼(CTA)을 생성하지 말 것.
- 텍스트는 최소화하며, 모든 섹션은 '작업물'이 주인공이 되도록 설계할 것.
